# Mebelson: Гид по стилю 2026 (Landing Page)

Проект представляет собой современную посадочную страницу для демонстрации электронных каталогов мебельной фабрики Mebelson. Реализован как SPA-приложение на React с интеграцией 3D-технологий просмотра документов.

## Технологический стек
- **Frontend:** React 19, Tailwind CSS.
- **AI Integration:** Google Gemini API (модель gemini-3-flash-preview) для работы бизнес-ассистента.
- **3D Flipbook:** Библиотека [DearFlip (dFlip)](https://js.dearflip.com/) для реалистичного листания PDF.
- **Backend (Target):** 1С-Битрикс (интеграция через JSON-объект в глобальной области видимости).

## Основные компоненты
1. **Hero-секция:** Имиджевый блок с акцентом на коллекции 2026 года.
2. **CatalogExplorer:** Сетка основных каталогов с фильтрацией (в перспективе) и кнопками быстрого просмотра/скачивания.
3. **LeafletsSection:** Блок рекламных листовок для вдохновения клиентов.
4. **CertificatesSection:** Раздел доверия с документами соответствия ГОСТ и экологическим стандартам.
5. **AIConsultant:** Интеллектуальный чат-бот, помогающий дилерам и покупателям ориентироваться в ассортименте.

## Архитектурные особенности
### 1. Интеграция DearFlip
Для активации 3D-просмотра необходимо:
- Купить лицензию на 3 сайта на официальном ресурсе.
- Подключить `dflip.min.js` и соответствующие CSS файлы на странице.
- Метод `openCatalog` в `App.tsx` передает URL файла в компонент `FlipbookDemo`, который инициализирует плагин.

### 2. Динамические данные
Приложение спроектировано для получения данных извне. Разработчику Bitrix необходимо подготовить JSON-объект `window.mebelsonData`, содержащий массивы объектов для каталогов, листовок и сертификатов.

### 3. Настройка AI (Gemini)
Для работы ассистента используется API ключ, передаваемый через переменную окружения `process.env.API_KEY`. В системе Битрикс это может быть реализовано через проброс ключа в JS при сборке или через проксирующий PHP-скрипт для безопасности.

## Администрирование
Все управление контентом осуществляется через административную панель Битрикс (Инфоблоки). 
- **Загрузка каталога:** Добавление элемента в инфоблок "Каталоги" с прикреплением PDF-файла.
- **Обновление сертификатов:** Замена файлов в соответствующем инфоблоке мгновенно обновляет их на странице.

## Требования к окружению
- Node.js 18+ (для сборки проекта).
- Наличие подключенного jQuery (требуется для текущей версии DearFlip).
- Браузеры с поддержкой WebGL для корректной работы 3D-анимации листания.
