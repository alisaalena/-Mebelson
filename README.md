# Mebelson: Гид по стилю 2026 (Чистый прототип для Bitrix)

Данный репозиторий содержит визуальный эталон верстки посадочной страницы для мебельной фабрики Mebelson. Проект полностью очищен от внешних зависимостей и готов к переносу в среду 1С-Битрикс.

## Статус проекта: ЭТАЛОН ВЕРСТКИ
- БЕЗ AI: Все функции искусственного интеллекта удалены.
- БЕЗ CDN: Шрифты и стили загружаются из локальных ресурсов (системный стек).
- БЕЗ ТРЕКЕРОВ: В коде отсутствуют счетчики аналитики и внешние пиксели.
- CMS Ready: Все пути к файлам заменены на плейсхолдеры-маркеры.
- ZERO EXTERNAL: Отсутствуют ссылки на любые внешние домены (протоколы http/https вырезаны).

## Технологический стек
- Frontend: React 19.
- CSS: Tailwind CSS (классы сохранены в JSX для PostCSS-сборки в Bitrix).
- 3D Flipbook: Библиотека DearFlip (требует локального подключения в Bitrix, официальный сайт плагина для покупки лицензии).

## Инструкция по интеграции в 1С-Битрикс

### 1. Подготовка Инфоблоков
Создайте инфоблоки с соответствующими полями:
- Каталоги: Название, Файл (PDF), Обложка, Год.
- Листовки: Название, Описание, Файл (PDF), Превью.
- Сертификаты: Название, Файл.

### 2. Подключение DearFlip
- Скачайте купленную версию DearFlip.
- Разместите папку скриптов по локальному пути вашего шаблона.
- Подключите ассеты в header.php.

### 3. Перенос верстки
- Используйте Tailwind-классы из компонентов как базу для настройки PostCSS в вашем проекте.
- Замените маркеры PLACEHOLDER_... в коде на реальные переменные из массива данных Bitrix (рекомендуется передавать данные через window.mebelsonData).

## Структура данных (JSON)
Ожидаемый формат данных для инициализации страницы:
```javascript
window.mebelsonData = {
  catalogs: [
    { id: 1, title: '...', pdfUrl: '/upload/...' }
  ]
};
```

## Проверка безопасности
В коде отсутствуют вызовы к внешним шрифтам, библиотекам генеративного ИИ или фотостокам. Проект полностью автономен.
